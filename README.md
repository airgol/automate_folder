- Execute the python file:  
`python3 automation.py`

- To schedule you python script in linux with *crontab* execute the next commads:  
`crontab -e`
- Add the nex line to crontab menu (Ex. Everyday at 18:00):  
`0 18 * * * python3 YOUR_PATH`
